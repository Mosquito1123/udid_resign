# udid_resign

```bash
ruby udid_resign.rb -d 00008020-00117C190C68002E -i ../glt_input.ipa

ruby udid_resign.rb -d 00008020-00117C190C68002E -i ../glt_input.ipa -o ../glt_output_resigned.ipa
```

**-d, --udid UDID**

> Device UDID 
> 用户设备唯一识别码 UDID


**-i, --input Input**

> Ipa Input Path
> ipa包的输入路径



**-o, --output Output**

> Ipa Output Path
> ipa包的输出路径

**-b, --bundleid BundleID**

> Pass-in Bundle Identifier
> 输入新的Bundle Identifier

**-u, --username Username**

> Apple ID
> Apple ID 账号

**-p, --password Password**

> Password Of Apple ID
> Apple ID 密码

**-n, --appname AppName**

> Pass-in App Name
> App 名称

**-N, --devicename DeviceName**

> Pass-in Device Name
> 设备名称
